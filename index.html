ROBS is a tool for very niche cases in OBS

ROBS is a controller for OBS built around the OBS WebSocket protocol. It lets you control OBS from the same machine or remotely, so more than one person can operate the same OBS session at the same time.

It interacts with OBS sources whose name starts with _.

It is most useful for mid to large productions where a lot of things need to be automated, where there isnt a lot of staff, and maybe you dont want to use actual proffesional tools.

Similar system of scripts and plugins that can be found in OBS, but with the advantage of being able to use them remotely, having more freedom to create interfaces, and everything a javascript file can do.

Quick Start

  1. Enable the OBS WebSocket server in Tools → WebSocket Server Settings.
  2. Launch ROBS and open the Config panel to set the host, port, and password.
  3. Save the settings. The dashboard lists every scene and only sources whose names start with _.
    4. Drop a plugin into the plugins folder and click its sidebar button to run it.
