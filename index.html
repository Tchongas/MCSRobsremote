What is ROBS?

ROBS is an OBS remote controller designed for livestream operators who need more than the default tools. It connects to OBS over WebSocket and exposes a clean dashboard where you can toggle sources, switch scenes, and trigger custom automation through JavaScript plugins.

Connect to OBS instances over the network via WebSocket and control them from a single window. Only sources prefixed with _ appear in the dashboard, keeping your workspace clean. Write JavaScript plugins to update text sources, sync runner data, or build custom workflows. Separate preview and program scenes with a single "Go Live" transition.

Key Features

  • Real-time scene and source synchronization across multiple users.
  • Studio Mode with preview/program separation and a sliding transition bar.
  • Plugin system loaded from the robs/plugins folder in your Roaming directory.
  • Built-in console and configuration UI for quick setup.
  • Debug menu accessible via a configurable hotkey.

Quick Start

  1. Enable the OBS WebSocket server in Tools → WebSocket Server Settings.
  2. Launch ROBS and click the Config button to set the host, port, and password.
  3. Save the settings and watch the dashboard populate with your _-prefixed sources.
  4. Drop a plugin into robs/plugins and hit the sidebar button to run it.
