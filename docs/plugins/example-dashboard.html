Example: Dashboard

This example shows how to build a plugin that reacts to the current dashboard state and updates OBS sources.

Scenario

You want a button that toggles a lower-third graphic for the current runner. The plugin reads the active scene, finds the lower-third source, and toggles its visibility.

Code

const { getCurrentScene } = require('../../renderer/logic/scenes');
const { pluginutils } = require('../../pluginutils');

function toggleLowerThird() {
  const scene = getCurrentScene();
  const sourceName = '_lower_third_' + scene;
  pluginutils.setSourceVisibility(sourceName, true);
}

module.exports = { toggleLowerThird };

Result

Clicking the sidebar button shows the lower-third for the current scene. Adapt the source name pattern to match your own naming convention.
