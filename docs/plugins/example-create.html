Example: Syncing Text Sources from a File

A common pattern: read a CSV, parse the data, and push values into OBS text sources. This is what the bundled RunnerSyncPlugin does.

Scenario

You have a runners.csv file in your plugin folder with runner names and times. You want a button that reads the file and updates the corresponding OBS text sources.

Code

// plugin.js
(function() {
  window.PluginUtils.registerSidebarButton(
    'RunnerSync',
    'sync-runners',
    'Sync Runners',
    async () => {
      const csv = await window.pluginAPI.readFile('runners.csv');
      const lines = csv.trim().split('\n');
      const headers = lines[0].split(',').map(h => h.trim());

      for (let i = 1; i < lines.length; i++) {
        const values = lines[i].split(',').map(v => v.trim());
        const row = Object.fromEntries(headers.map((h, j) => [h, values[j] || '']));
        const slot = i; // 1-based

        await window.PluginUtils.setTextSource(`_Runner${slot}Name`, row.name || '');
        await window.PluginUtils.setTextSource(`_Runner${slot}Time`, row.time || '');
      }

      window.uiHelpers.logSuccess('Runners synced', 'RunnerSync');
    }
  );
})();

CSV Format

name,time
PlayerOne,1:23:45
PlayerTwo,1:31:00

The source names need to exist in OBS and start with _. The plugin does not create them, it just sets their text.
