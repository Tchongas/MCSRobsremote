Example: Create

This example shows how to create and update OBS sources from a plugin.

Scenario

You want a button that creates a new text source for each runner in a list. The plugin reads a CSV, creates the sources if they do not exist, and sets their text.

Code

const { pluginutils } = require('../../pluginutils');

function createRunnerSources() {
  const runners = pluginutils.readFile('runners.csv').split('\n');
  runners.forEach((runner, index) => {
    const sourceName = `_runner_${index + 1}`;
    pluginutils.createTextSource(sourceName, runner);
  });
}

module.exports = { createRunnerSources };

Result

Clicking the button creates missing text sources and updates them. You can then use these sources in your scenes.
