Example: Automation

This example shows how to automate a sequence of OBS actions with a single plugin button.

Scenario

You want to run a commercial break: mute the microphone, show a "BRB" scene, and start a timer. One plugin can do all three in order.

Code

const { pluginutils } = require('../../pluginutils');

function startCommercialBreak() {
  pluginutils.setInputMute('Microphone', true);
  pluginutils.setCurrentProgramScene('BRB');
  setTimeout(() => {
    pluginutils.setInputMute('Microphone', false);
    pluginutils.setCurrentProgramScene('Live');
  }, 120000);
}

module.exports = { startCommercialBreak };

Result

Clicking the button mutes the mic, switches to the BRB scene, and returns after two minutes. Adjust the timing and scene names to match your setup.
