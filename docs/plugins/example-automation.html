Example: Automation

A sidebar button that runs a sequence of OBS actions in order. No user interaction needed beyond clicking the button.

Scenario

You are between races and want a single button that: mutes the microphone, swaps two camera sources, and then switches the scene to an intermission layout.

Code

// plugin.js
(function() {
  window.PluginUtils.registerSidebarButton(
    'IntermissionPlugin',
    'start-intermission',
    'Intermission',
    async () => {
      const SCENE = 'Intermission';

      // Mute the mic
      await window.obsAPI.sources.setMute('_Microphone', true);

      // Swap camera positions in the current scene
      await window.PluginUtils.swapSourcePositions('_CamA', '_CamB', SCENE);

      // Switch to intermission scene
      await window.obsAPI.scenes.change(SCENE);

      window.uiHelpers.logSuccess('Intermission started', 'IntermissionPlugin');
    }
  );
})();

Reading a CSV for Data-Driven Automation

If you need to drive the automation from a file (like runner data), read it with pluginAPI.readFile: 

const csv = await window.pluginAPI.readFile('runners.csv');
const rows = csv.trim().split('\n').map(l => l.split(','));
// rows[0] = headers, rows[1..] = data
