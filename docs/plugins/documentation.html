Plugin Documentation

Plugins are JavaScript files that export functions. The runtime turns each exported function into a sidebar button.

Basic Plugin

A minimal plugin exports a function and optionally uses pluginutils helpers.

function helloPlugin() {
  // Your plugin logic here
}

module.exports = { helloPlugin };

pluginutils

Common helpers available to plugins:

  • pluginutils.readFile(path) — read a file from the plugin directory.
  • pluginutils.fetchJSON(url) — fetch and parse JSON.
  • pluginutils.setTextSource(name, value) — update a text source in OBS.
  • pluginutils.setBrowserSourceURL(name, url) — update a browser source URL.
  • pluginutils.getSources() — access the current OBS sources list.

Folder Packages

Advanced plugins can be packaged as folders with a plugin.js entry point plus resources like Data.csv, README.md, and configuration files.
