Connection

ROBS connects to OBS through the OBS WebSocket server. Multiple users can connect to the same OBS session locally or over the network.

Enable the WebSocket Server

OBS 28 and newer include the WebSocket server by default. To enable it:

  1. In OBS, open Tools → WebSocket Server Settings.
  2. Check Enable WebSocket server.
  3. Set a password if you want one.
  4. Note the Server Port (default is 4455).

Configure ROBS

Launch ROBS and open the config panel:

  1. Enter the Host — localhost if OBS is on the same machine, or an IP address for a remote PC.
  2. Enter the Port from the OBS settings.
  3. Enter the Password if you set one.
  4. Click Save. The connection status appears in the sidebar.

Troubleshooting

  • Make sure OBS is running before ROBS tries to connect.
  • Check that the firewall allows the chosen port.
  • For remote connections, confirm both devices are on the same network or route the port correctly.
