Dashboard

The dashboard is the main control surface in ROBS. It lists scenes and the sources you have chosen to control.

Scene List

The scene list is on the left. Click a scene to select it, then click Go Live to make it live. Double-click a scene to go live immediately.

Source Dashboard

Only sources whose names start with _ appear in the dashboard. This keeps the workspace focused on the sources you want to control. Each row has a toggle to enable or disable the source.

Transition Bar

When a selected scene is not yet live, the Go Live bar appears. Click it to transition.
