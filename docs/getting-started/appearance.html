Appearance

All appearance settings are in Settings → Appearance. Changes apply immediately and are saved automatically.

Hiding Panels

You can hide any section you do not need:

  • Hide Console — removes the log panel at the bottom.
  • Hide Stream Controls — removes the stream start/stop buttons, good if you dont want to accidentally stop the stream.
  • Hide Dashboard — removes the source control area entirely.
  • Hide Scenes — removes the scene list.
  • Hide Quick Actions — removes the quick action buttons.

Plugin-Only Mode

Hides everything except the plugin sidebar. Useful if you only need your plugin buttons and want a minimal window, This pretty much turns ROBS into a macrodeck.
