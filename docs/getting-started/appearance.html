Appearance

ROBS supports a light and dark theme. The interface is built to be compact and readable for livestream operators.

Theme Toggle

Click the moon/sun icon in the top-right corner to switch between light and dark mode. The choice is saved in local storage and applies on the next launch.

Layout

The main window is split into three areas: the scene list, the source dashboard, and the sidebar with plugin buttons. The bottom panel holds the config controls. Resize the window freely; the dashboard adapts to the available space.

Future Customization

More appearance options (font size, accent color, panel density) may be added later. For now, the theme toggle covers the main visual preference.
