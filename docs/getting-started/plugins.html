Plugins

Plugins add buttons to the ROBS sidebar and automate OBS actions. This page covers how to install and run them.

Plugin Folder

Plugins are loaded from the plugins folder inside the robs directory in your Roaming folder. On Windows this is usually %APPDATA%\robs\plugins.

Install a Plugin

  1. Download or write a plugin .js file.
  2. Copy it into the robs/plugins folder.
  3. Restart ROBS. The plugin button appears in the sidebar.

Run a Plugin

Click the plugin button in the sidebar. The plugin runs its function and can update OBS sources, read files, or trigger any other automation.

Bundled RunnerSyncPlugin

ROBS ships with RunnerSyncPlugin, which reads runner data from a CSV and updates text sources. It is a good example of what a plugin can do.
