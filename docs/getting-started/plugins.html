Plugins

Plugins are JavaScript files that add functionality to ROBS. They can add buttons to the sidebar, add options for sources, fetch data, run automations, similar to how OBS scripts work.

Plugin Folder

Plugins live in %APPDATA%\robs\plugins. You can open this folder from Settings → Plugins. Each plugin is a folder containing a plugin.js file and any data files it needs.

Screenshot: Plugins tab in settings showing loaded plugins

Installing a Plugin

  1. Drop the plugin folder into %APPDATA%\robs\plugins.
  2. ROBS detects the change and reloads automatically.
  3. In certain cases you might need to restart ROBS for the plugin to start working.

Using a Plugin

Plugins can have multiple functions:

  • Function buttons in the sidebar - You can click them to trigger the actions.
  • Create.js Plugins BETA - You can go in the Create tab in settings, and run their function to add the plugins sources in your OBS, usually in a new scene for you to copy and paste
  • Plugins Config and README - They exist in the plugin tab in the settings page, you can click the buttons to edit them, if there are no buttons it means the file doesnt exist, as its not necessary.
  • Dashboard Plugins - They can add something for a source handler, like a text formatter button for all text sources.
  • Workspace Plugins BETA - Workspaces are popout screens that you can add your own HTML and CSS to create custom interfaces for controling OBS from within ROBS.
