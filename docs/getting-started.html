Getting Started

You need OBS Studio 28 or newer with the WebSocket server turned on. That is it for prerequisites. The rest is just pointing ROBS at the right address.

Topics

  • Connection — local setup, remote setup, profiles.
  • Dashboard — how scenes and sources work in ROBS.
  • Appearance — scaling, hiding panels, plugin-only mode.
  • Plugins — how to install and run plugins.

Troubleshooting

  • OBS needs to be running before ROBS tries to connect.
  • If you are on OBS older than 28, you need to install the obs-websocket plugin manually.
  • For remote connections, check that your firewall is not blocking the WebSocket port.
