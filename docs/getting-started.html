Getting Started

This page is the starting point. The detailed steps for each part of the setup are in the subtopics below.

The Basics

You need OBS Studio with the WebSocket server enabled. OBS 28 and newer include it by default. If you are on an older version, install the WebSocket plugin first.

The app was made and tested in windows, but as it is an electron App, it shouldnt be hard to make it work on other platforms too.

Troubleshooting

  • Make sure OBS is running before ROBS connects.
  • Check that the firewall is not blocking the WebSocket port.
  • For remote connections, confirm both devices are on the same network or the port is routed correctly.
