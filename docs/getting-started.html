Getting Started

ROBS runs as a desktop application built with Electron. This guide covers the prerequisites, how to connect to OBS, and how to use the dashboard.

Prerequisites

  • OBS Studio with the WebSocket Server plugin enabled (OBS 28+ includes it by default).
  • A local network path or localhost connection between ROBS and the OBS machine.
  • Node.js if you plan to run the app from source or develop plugins.

Connect to OBS

Open the OBS WebSocket server settings:

  1. In OBS, go to Tools → WebSocket Server Settings.
  2. Enable the server and set a password if required.
  3. Note the Server Port (default is 4455).

Then configure ROBS:

  1. Launch ROBS and click the Config button in the bottom-left corner.
  2. Enter the host (localhost or an IP address), port, and password.
  3. Click Save. The connection status will update in the sidebar.

Dashboard Overview

The dashboard shows every scene and all sources whose names start with _. You can click any source row to toggle its visibility or open its parameters. The console at the bottom logs connection events and remote actions.

Studio Mode

ROBS uses Studio Mode by default:

  • Preview (red indicator) is the scene you are editing.
  • Program (blue indicator) is the live scene your viewers see.
  • Click a scene to set it as preview. Click the Go Live button to transition it to program.
  • Double-click a scene to select it and go live immediately.

Keyboard Shortcuts

  • Ctrl + Enter — Transition the current preview scene to program.
  • Debug menu hotkey — Configured in settings; opens the developer debug panel.
