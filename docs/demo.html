Demo

ROBS is a desktop Electron app. The best way to see it is to run it. The screenshots below show the main interface ideas.

Dashboard & Scene Control

The main window shows the scene list on the left, the source dashboard in the center, and the sidebar with plugin buttons on the right. The bottom panel has the config controls.

Plugin Workflow

Plugins appear as buttons in the sidebar. The bundled RunnerSyncPlugin reads a CSV, fetches runner data, and updates text sources in one click.
