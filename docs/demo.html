Demo

Below is a preview of the ROBS interface. Because ROBS is a desktop Electron application, the live experience is best seen by running the app. The screenshots and workflow descriptions here show the main interface concepts.

Dashboard & Scene Control

The main window shows the connected scene list on the left, the source dashboard in the center, and the sidebar with plugin buttons on the right. The bottom panel displays the console and configuration controls.

Studio Mode Transition

When the preview scene differs from the live scene, a red "Go Live" transition bar slides in. Click it or press Ctrl + Enter to push the preview scene to program.

Plugin Workflow

Plugins appear as buttons in the sidebar. The bundled RunnerSyncPlugin reads a CSV, fetches runner data, and updates text sources in one click.
